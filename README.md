# 🧮 Calculadora con MongoDB Atlas

⭐ **5to Semestre - UPC | Julio 2026**

Esta aplicación web, que se basa en una calculadora matemática, es una prueba práctica para desarrollar una web Fullstack utilizando una arquitectura simple y funcional. Su objetivo principal es realizar cálculos matemáticos en tiempo real y guardar automáticamente cada operación en un historial persistente en la nube, permitiendo consultar las operaciones previamente realizadas mediante una interfaz intuitiva y responsiva.

---

## 🛠️ Herramientas y Tecnologías

* **Frontend:** HTML5, CSS3 (Flexbox), JavaScript Vanilla (`fetch`).
* **Backend:** Node.js, Express (Servidor en puerto `5000`), Live Server (Puerto `5501`).
* **Base de Datos:** MongoDB Atlas (Nube), Mongoose (ODM), `.env` (Variables de entorno).
* **Librerías (NPM):** `express`, `mongoose`, `dotenv`, `nodemon`.
